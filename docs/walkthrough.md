# 操作指南 - 新版AIS与雷达匹配工作流

我已重构您的代码以支持**批量处理**和**交互式可视化**。以下是如何使用新系统。

## 1. 准备数据（如果您已有 .mat 文件，此步可选）
如果您需要从SQL重新提取数据，请运行改进后的脚本：
-   `SQL2MAT_radar.m`
-   `SQL2MAT_AIS.m`

## 2. 运行批量匹配
运行新脚本以自动匹配**所有**UID到MMSI。
```matlab
RunBatchMatching
```
-   **预期结果**：进度条将显示匹配进度。
-   **输出**：将创建一个名为 `MatchResults.mat` 的文件，并在命令窗口打印摘要表。

## 3. 查看结果
运行查看器工具浏览匹配结果。
```matlab
ViewTrajectories
```
-   **交互**：工具将告诉您找到了多少个匹配结果（例如，“共有 50 个匹配结果”）。
-   **输入**：输入一个数字（例如 `1`，`2`，`10`）以查看该特定匹配。
-   **输出**：将弹出图形窗口，显示雷达轨迹（蓝色）与AIS轨迹（红色）。

## 4. 主要改进
-   **不再需要手动输入**：您无需再复制粘贴UID。
-   **效率**：一次匹配所有数据，随时查看。
-   **可读性**：所有新代码均有中文注释。
-   **标准化**：数据加载脚本现在保持一致。
